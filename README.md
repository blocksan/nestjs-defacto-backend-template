## Description

It is a nestjs based template which has features already plugged in to start the rapid development of any backend project
1. Custom Logger
2. Swagger
3. Passport Authentication
4. JWt based authentication
5. Authrization Gaurd
6. Typeorm integration with Mariadb
7. User entity
8. Test component (Above implementation with examples)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
